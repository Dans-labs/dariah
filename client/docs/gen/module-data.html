(static) getData(sources, component, notification) → {Promise}

Source:
  • lib/data.js, line 37

A component may request for a number of data queries in one go. Every query is specified as Source object in the parameter sources.

The expected response is always a json Result object.

Parameters:
Name Type Description
sources Array.<Source>

List of data sources to fetch
component Component

Component for which to fetch data
notification Object

The Notification component that receives error/progress messages
Returns:

This is an asynchronous call. The fetched data comes in as a promise, to with getData attaches .then handlers, which will be new promises. If the fetch is successful, the resolved data will added to the state[branch] of component. Progress and error messages from the server appear in the fetched json as messages. If the fetch itself failed, we handle it by transforming the error into a message. All messages generated in the process will be send to the notification component.
